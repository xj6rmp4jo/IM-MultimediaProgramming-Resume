演算法分析　渡河問題

　　有 M 隻狼與 N 隻羊需從西岸渡河到東岸，其使用的船需乘載至少一隻才渡河，最多只能乘載兩隻。由於狼會吃羊的緣故，每次渡河時，不管在西岸或東岸，羊的數目均不能比狼少。(遞迴)

演算法分析　最大子陣列問題

　　給定一整數陣列，其中可能包含正或負整數，目的是找到子陣列，即連續元素整數和，且其總和最大。(Divide and Conquer)

演算法分析　LCS 問題

　　給定兩個序列(或字串)，目的是找到兩個序列連續的共同字元，稱為共同序列，且其長度必須最長。(動態規劃法)

演算法分析　推箱子遊戲

　　使用者以文字輸入二維迷宮圖，其中包含角色、箱子、目的地、障壁...等，目的是以最小步數將箱子搬運至目的地。(遞迴)

畢業專題　基於街景分析及揮手辨識之智慧公車安全輔助系統

　　建構出一個具街景分析和揮手辨識功能的導航系統，使駕駛能夠獲取比起一般 2D 圖像導航更豐富的道路環境資訊。(機器學習、語義分割、手勢辨識、Non blocking socket)

作業系統　多工處理

　　比較以下四種方法之執行時間並分析之 :

  1. 將 N 筆數目字直接進行 BubbleSort。
  2. 將 N 筆數目字切成 K 份，先在 1 個 process 內對 K 份資料進行 BubbleSort 之後，再用同一個 process 作 MergeSort。
  3. 將 N 筆數目字切成 K 份，並由 K 個 processes 各別進行 BubbleSort 之後，再用 K-1 個 process(es) 作 MergeSort。
  4. 將 N 筆數目字切成 K 份，並由 K 個 threads 各別進行 BubbleSort 之後，再用 K-1 個 thread(s) 作 MergeSort。
